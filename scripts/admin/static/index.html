🚀 Cómo publicar tus cambios

Esta app guarda directamente en los archivos .cho del repositorio. Cuando hayas terminado de editar, haz un git commit y un git push a la rama. Un GitHub Action se encarga automáticamente de regenerar songs-vX.json y subirlo a Firebase.

En la terminal, desde la raíz del repo: git add songs/ && git commit -m "..." && git push

📐 Importar de LaTeX

Archivos .tex en scripts/input/<categoría>/. Si el título coincide con una canción ya existente en el repo, se ofrece ↻ sobrescribir manteniendo número en lugar de crear otra. Tras importar, el .tex se mueve a scripts/input/processed/.

LaTeX tiene prioridad sobre el cantoral DOCX — si una canción está en ambos, mejor importarla desde aquí (los acordes vienen ya bien posicionados).

🔀 Reordenar canciones

    
Elige una categoría, arrastra para reordenar y pulsa Aplicar. Los archivos
       se renumerarán (01., 02., …) manteniendo el slug. Se hace backup en
       songs-backup-edits/.

    

      
      
      
    


    
    
      
    

  


  
  

    
🗂 Gestión de backups

    
Cada vez que guardas o borras una canción se crea una copia en
       songs-backup-edits/. Aquí puedes limpiarlos manualmente cuando ya no los necesites.

❓ Atajos y trucos

        
      

      

        

          
Editor visual

          

            
Arrastrar acordeMueve el acorde. Durante el drag verás en azul la letra donde caerá y una etiqueta del modo (PALABRA/SÍLABA/LIBRE).

            
(sin modificador)Snap al inicio de sílaba más cercano (silabeado español).

            
Shift al soltarSnap a inicio de palabra.

            
Alt al soltarSin snap — colocación libre carácter a carácter (pixel-perfect).

            
Hover sobre líneaMuestra puntos grises (·) en los inicios de sílaba y barras azules en los inicios de palabra, para ver visualmente dónde puedes anclar.

            
Doble-click acordeEdita texto del acorde (vacío = borrar).

            
Click derecho acordeBorra con confirmación.

            
SuprBorra el acorde seleccionado.

            
Doble-click en letraEdita el texto de esa línea entera (los acordes se reubican por palabra).

            
Click en margen izq.Selecciona la línea (azul). Útil para marcar estribillos o copiar acordes.

            
Shift+click en margenSelección de rango.

            
Ctrl/Cmd+SGuarda la canción.

          

          
Marcar estribillo

          
    
            
  1. Selecciona las líneas (clicks en el margen izquierdo).
    2.

Copiar / pegar estribillo

          
Botón "Insertar estribillo" → elige cuál de los estribillos ya marcados quieres pegar; se inserta justo después de la línea actualmente seleccionada (o al final).

          
Copiar patrón de acordes entre estrofas

          
    
            
  1. Selecciona las líneas de la estrofa con los acordes que quieres copiar.
    2. 
            
  2. Pulsa "Copiar acordes".
    3. 
            
  3. Selecciona las líneas destino (sin acordes o con pocos).
    4. 
            
  4. Pulsa "Pegar acordes". Se alinean por palabra: el acorde N de la línea origen va a la palabra N de la línea destino. Luego ajusta a mano lo que haga falta.
    5.
